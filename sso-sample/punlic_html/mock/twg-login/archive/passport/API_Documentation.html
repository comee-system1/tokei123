認可エンドポイント  ¶

アクセストークンを取得するための認可コードをURLパラメーターで返します。

認可コードの取得
GET/oauth/authorize{?client_id,response_type,scope}

Example URI

GET /oauth/authorize?client_id=57297408867&response_type=code&scope=*
URI Parameters
HideShow
client_id
number (required) Example: 57297408867

発行されたクライアントID

response_type
string (required) Example: code

トークン発行方法（codeを指定します。）

scope
string (required) Example: *

取得したいユーザー情報

redirect_uri: `https://your-site.com/social_login/callback`
string (required) 

クライアント作成時に指定したリダイレクトURL

Response  200
HideShow

    • hint エラーを解消するためのヒント
    • message エラーメッセージ
